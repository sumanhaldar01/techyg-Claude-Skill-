# Claude-Skill-techyg Smart Tech Recommendation Engine
TechYG is a Claude-skill  that provides intelligent recommendations for tech products and home appliances based on user needs, budget, and real-world data.  It goes beyond basic suggestions by combining structured analysis models with live research.
## 🎯 Problem
People rely on:
* Random YouTube reviews
* Biased affiliate blogs
* Conflicting opinions

This leads to poor buying decisions.

## 💡 Solution

TechYG simulates a data-driven decision process using:

* 🔍 Web Research (prices, reviews, benchmarks)
* ⚖️ Weighted Scoring Model (dynamic based on use-case)
* 📊 Quantitative + Qualitative Analysis
* 💰 Value for Money Evaluation
* ⚠️ Risk Analysis (heating, QC, service issues)
* 📉 Benchmark vs Real-world Performance Gap

## 🧠 How It Works

### Input

* Natural language query
  Example: *"Best gaming phone under ₹50K"*
* Optional: Images / documents (spec sheets, comparisons)

### Process

1. Intent + budget detection
2. Category & use-case classification
3. Data aggregation
4. Weighted scoring
5. Risk + reality checks

### Output

* Top 5 recommendations (ranked)
* Category-based suggestions (e.g., gaming vs camera)
* Pros / Cons
* Best For / Avoid If
* Final verdict

## 📦 Example Use Cases

* Smartphones (gaming, camera, all-rounder)
* Laptops (coding, editing, gaming)
* ACs, TVs, appliances
